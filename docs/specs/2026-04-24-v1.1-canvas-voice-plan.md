## 风险 & 备选

| 风险 | 应对 |
|---|---|
| MeloTTS 中文长稿效果不够好 | 换 Replicate IndexTTS-2（+ 账号，¥0.7/月）— 只换 `internal/audio/tts_*.go` 一个 client |
| CF 免费额度用尽 | 理论需每天 107 次才超，不会 |
| X6 CDN 被墙 | jsDelivr + unpkg + cdnjs 多 CDN fallback |
| LLM 输出非法 JSON | 重试 backoff + 最终兜底 fallback flow schema |
| 分支隔离破损（意外切回 main） | 每次 Bash 之前先 `git -C /root/briefing-v3 branch --show-current` 验证 |
